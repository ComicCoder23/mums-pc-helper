PC HELPER — START HERE


Purpose
This pack is for working on the PC in a simple, controlled way without duplicating work or overbuilding the system.


Main rule
Use the existing docs only.
Do not create extra versions unless something is genuinely missing.


Working order
1. Open PC Helper - First Visit Checklist
2. Open PC Helper - Cleanup Order
3. Keep notes during the session
4. Only use tools or scripts if they are actually needed


What this pack is for
- checking what is slowing the PC down
- cleaning obvious junk safely
- organising the visit in the right order
- keeping clear notes of what was moved, deleted, or left alone


What not to do
- do not delete lots of files in one go
- do not remove anything you are unsure about
- do not install lots of utilities unless the built-in Windows tools are not enough
- do not make duplicate copies of the same instructions


Session principle
Simple first:
- check access
- ask what the problem is
- clean Downloads first
- then Desktop
- then review Documents/Pictures/Videos only if needed
- empty Recycle Bin last


Core docs in this pack
- PC Helper - First Visit Checklist
- PC Helper - Cleanup Order


What to record during the visit
- what feels slow
- what was deleted
- what was moved
- what still needs checked later
- any program that looks suspicious, unused, or confusing


End goal
Leave the PC clearer, lighter, and easier to use, while keeping a record of what was done.
