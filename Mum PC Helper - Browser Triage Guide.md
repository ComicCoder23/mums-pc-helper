MUM PC HELPER — BROWSER TRIAGE GUIDE


Purpose
Use this guide when the PC problem sounds like browsing rather than storage.
This helps you separate browser clutter from wider system problems.


Common browser complaints
- Facebook is slow
- pages take ages to open
- too many pop-ups
- strange home page or search page
- too many tabs open all the time
- browser feels messy or confusing
- browser opens unwanted pages


Quick browser check
1. Ask which browser Mum actually uses
2. Check how many tabs are open
3. Check whether the browser opens to a strange page
4. Check whether there are lots of toolbars or extensions
5. Ask if pop-ups happen on all websites or only certain ones


Fast safe actions
1. Close obvious extra tabs
2. Restart the browser
3. Check extensions
4. Remove only obvious junk extensions
5. Check browser home page and default search engine
6. Clear browsing cache only if the browser is slow or glitchy


Do not do immediately
- do not sign Mum out of everything unless necessary
- do not reset the whole browser without warning
- do not remove extensions you do not understand if they may be needed
- do not clear saved passwords or autofill without asking


How to read the problem


Case A — Browser slow but PC otherwise feels okay
Likely cause:
- too many tabs
- too many extensions
- bloated cache


Action:
- reduce tabs
- review extensions
- clear cache if needed


Case B — Pop-ups and strange pages
Likely cause:
- bad extension
- browser setting change
- unwanted software


Action:
- check extensions first
- check homepage and search engine
- note suspicious behavior in Session Notes
- use malware scan only if needed


Case C — Everything online feels slow
Likely cause:
- internet issue
- too many background tabs
- browser heavy usage
- system slowness, not just browser


Action:
- test another site
- check Wi-Fi speed practically
- compare browser issue versus whole-PC issue


Case D — One website is bad, others are fine
Likely cause:
- site-specific problem
- account issue
- browser compatibility issue


Action:
- do not over-clean the whole PC
- test another browser only if necessary


Extension review rule
Remove an extension only if:
- it is clearly junk
- Mum does not use it
- it looks suspicious
- it matches the pop-up or hijack behavior


Leave it alone if:
- it is a password manager
- it is security-related
- it is needed for accessibility or printing
- you are unsure


Cache clearing rule
Clear cache when:
- pages are loading strangely
- websites are stale or glitchy
- browser feels sluggish


Warn first:
- some websites may need sign-in again


Signs of hijack or junk behavior
- homepage changed by itself
- search engine changed unexpectedly
- extra ads everywhere
- random tabs opening
- lots of notifications from unknown sites


If that happens
- stop normal cleanup
- note the exact behavior
- check extensions and notification permissions
- use security tools only if clearly needed


Success looks like
- browser opens cleanly
- fewer tabs and less clutter
- no obvious junk extensions
- homepage and search look normal
- browsing feels simpler and faster


Rule
Treat browser problems as browser problems first.
Do not assume the whole PC needs major cleanup if the issue is only in the browser.
