MUM PC HELPER — CLEANUP ORDER


Use this order exactly unless there is a clear reason not to.


1. Confirm the problem
Before deleting anything, find out what is actually wrong.
Examples:
- low disk space
- too much clutter
- slow startup
- confusing files everywhere
- downloads pile-up


2. Check free space first
Look at available storage before doing cleanup so you can compare before and after.


3. Start with Downloads
This is the safest high-value cleanup area.
Delete only obvious junk:
- old installers
- duplicate downloads
- extracted zip leftovers
- temporary files
- unwanted PDFs and random downloads


4. Move to Desktop
Clean obvious clutter:
- duplicate shortcuts
- misplaced files
- things that belong in Documents, Pictures, or Videos


5. Review Recycle Bin last
Do not empty it until the main cleanup pass is finished.


6. Only check Documents if needed
Do not go digging through Documents unless there is a space problem or a clear clutter issue.
Be cautious here.


7. Only check Pictures and Videos if needed
These can contain important personal files.
Do not delete unless the owner is clear.
If unsure, leave them alone.


8. Programs review only after file cleanup
If the PC is still struggling, then review installed apps and startup items.
Do not uninstall things casually.


9. Use built-in tools first
Preferred tools:
- File Explorer
- Storage settings
- Disk Cleanup
- Task Manager
- Settings > Apps


10. Record what changed
Keep notes on:
- deleted items
- moved items
- folders reviewed
- anything left for later


Golden rules
- obvious junk first
- uncertain items later
- personal folders carefully
- system folders never
- no mass deletion
- no duplicate work


End of cleanup
- Empty Recycle Bin
- Check free space again
- Confirm whether the PC feels better
- Save notes for the next visit
