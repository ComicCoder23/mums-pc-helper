WHAT TO LOOK FOR
- low disk space
- full Downloads folder
- cluttered Desktop
- obvious old installers
- duplicate downloads
- zip files no longer needed
- startup slowness
- too many unknown programs


DO NOT TOUCH YET
- system folders
- unknown files
- printer software unless clearly unnecessary
- security software unless clearly unnecessary
- anything Mum says is important


SAFE RULE
Audit first.
Delete obvious junk only.
If unsure, leave it and write it down.

- clearer understanding of the PC
- no accidental deletion
- good session notes for next time
