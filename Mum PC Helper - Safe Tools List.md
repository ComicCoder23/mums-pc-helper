MUM PC HELPER — SAFE TOOLS LIST


Use built-in tools first.
Only move to extra tools if the built-in tools are not enough.


Tier 1 — Built-in Windows tools
1. File Explorer
- check Downloads
- check Desktop
- sort by size or date
- move or delete obvious junk only


2. Settings > System > Storage
- check free space
- identify where storage is being used


3. Disk Cleanup
- remove temporary files and safe junk
- do not guess at advanced system items if unsure


4. Task Manager
- check startup apps
- check what is using memory or CPU
- disable only obvious unnecessary startup items


5. Settings > Apps
- review installed programs carefully
- do not uninstall anything unless it is clearly unused and safe to remove


Tier 2 — Optional trusted tools
Use only if needed.


1. WizTree or WinDirStat
- helps find large folders fast
- use for visibility first, not instant deletion


2. Autoruns
- advanced startup review
- use only if startup clutter is a real issue


3. Malwarebytes Free
- use if there are pop-ups, suspicious behavior, or malware concerns


What not to start with
- registry cleaners
- aggressive PC boosters
- random driver updaters
- junk uninstallers that remove things automatically


Tool decision rule
- built-in first
- visibility before deletion
- notes before changes
- if unsure, do not use it yet


Quick default loadout
For most visits, this is enough:
- File Explorer
- Storage settings
- Disk Cleanup
- Task Manager


Only add more if the problem clearly needs it.
