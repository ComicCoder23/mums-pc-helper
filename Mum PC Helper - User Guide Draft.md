MUM MUM PC HELPER — USER GUIDE DRAFT


Purpose
This guide explains how to use Mum PC Helper safely as a simple first-pass Windows PC support product.


What this product is
Mum PC Helper is a guided support kit for checking a slow or cluttered Windows PC and taking the clearest safe actions first.


What this product is not
- not a deep repair suite
- not an aggressive cleanup tool
- not a malware removal toolkit
- not a promise to fix every PC problem in one visit


Main principle
Take the clearest safe win first.
If unsure, leave it alone.


BEFORE YOU START


What you need
- access to the PC
- permission from the user
- basic Windows access
- enough time to work slowly and safely


What to ask first
Before doing anything, ask:
- What feels slow?
- What do you use this PC for most?
- Is anything not working properly?
- Are there files you are worried about losing?
- Are there things you do not want touched?


If the user is worried about losing files
Be extra cautious.
Do not delete personal files unless they are clearly unwanted.


HOW TO USE THE PRODUCT


Best opening order
1. Open Start Here
2. Open Quick Run Card
3. Open Visit Workflow One Pager
4. Open Session Notes
5. Ask the opening questions


The default safe path
For most visits, use this order:
1. Run PowerShell Disk Space Audit
2. Run PowerShell Downloads Folder Audit
3. Clean obvious junk from Downloads
4. Tidy obvious Desktop clutter
5. Empty Recycle Bin
6. Recheck free space
7. Write Session Notes


Only go further if needed
- If startup is slow, review startup items
- If there are too many unknown apps, export installed programs
- If space is still tight, run the large files finder
- If the problem is browsing or pop-ups, use the browser triage guidance


WHAT THE MAIN SCRIPTS DO


1. Disk Space Audit
Use this first.
It checks:
- free space on local drives
- size of Desktop, Documents, Downloads, Pictures, and Videos


Use it when:
- the PC feels full
- the user says there is no space
- you want to know where to look first


2. Downloads Folder Audit
Use this when Downloads may be the clutter source.
It checks:
- large downloads
- old files
- likely installers
- zip and archive files


Use it when:
- the Downloads folder is usually messy
- the PC looks cluttered
- you want an easy cleanup win


3. Installed Programs Export
Use this only when needed.
It creates a list of installed programs for review.


Use it when:
- the PC feels cluttered or confusing
- there are many apps the user may not use
- you want a record before removing anything


4. Startup Items Review
Use this when startup is slow.
It creates a review list of startup items.


Use it when:
- the PC takes a long time to open after power-on
- too many background items may be starting automatically


5. Large Files Finder
Use this when space is still low after the basic cleanup.
It lists the largest files in safe user folders.


Use it when:
- space is still tight
- Pictures, Videos, or Downloads may contain very large files


HOW TO READ THE RESULTS


If free space is low
- start with Downloads
- then Desktop
- then Recycle Bin
- only then consider larger personal folders carefully


If Downloads has lots of old installers and archives
- likely easy cleanup win
- review with the user first
- remove only obvious leftovers


If startup items are numerous
- startup clutter may be contributing to slowness
- review carefully in Task Manager
- do not disable unknown or system-related items casually


If there are many unknown installed programs
- discuss them first
- do not uninstall based on names alone
- leave security, printer, and Windows-related items unless you are sure


If the browser is the main complaint
- treat it as a browser problem first
- check tabs, extensions, homepage, search engine, and pop-up behavior
- do not over-clean the whole PC if the issue is only in the browser


SAFE CLEANUP ORDER


Use this order unless there is a very clear reason not to:
1. Downloads
2. Desktop
3. Recycle Bin
4. Documents only if needed
5. Pictures and Videos only if needed
6. Installed apps only if needed
7. Startup items only if needed


WHAT IS USUALLY SAFE TO CLEAN


Often safe to review for removal:
- old installers
- duplicate downloads
- extracted zip leftovers
- obvious junk on the Desktop
- files the user confirms are not needed


WHAT NOT TO TOUCH CASUALLY


Do not touch casually:
- system folders
- unknown files
- printer software
- security software
- Windows-related components
- browser passwords or account data without asking


STOP RULES


Stop and leave it alone if:
- you are unsure what it is
- it looks important
- it looks system-related
- it looks security-related
- it looks printer-related and still in use
- the risk is higher than the likely gain


BROWSER TRIAGE


If the complaint is about browsing, pages loading slowly, or strange pop-ups:
- ask which browser is actually used
- check how many tabs are open
- check extensions
- check homepage and default search engine
- clear cache only if needed


Warning
Do not sign the user out of everything unless necessary.
Do not reset the whole browser without warning.


SESSION NOTES


During the visit, record:
- what was checked
- what was changed
- what was deleted
- what was moved
- what was left alone
- what needs looked at next time


END OF SESSION


Before finishing:
- check whether the main complaint improved
- recheck free space if storage was the issue
- save Session Notes
- write the next best step


SUCCESS LOOKS LIKE


A good session does not mean fixing everything.
A good session means:
- taking the clearest safe win
- making the PC a bit better
- avoiding damage
- keeping notes for later


FINAL RULE
Keep it simple.
Audit first.
Delete obvious junk only.
If unsure, leave it and write it down.
