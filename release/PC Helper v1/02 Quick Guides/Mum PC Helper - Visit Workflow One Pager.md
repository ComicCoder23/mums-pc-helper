PC HELPER — VISIT WORKFLOW ONE PAGER


Purpose
This is the compressed field workflow for a real visit.
Use this if you want the shortest safe version of the whole system.


START
1. Open Master Index
2. Open Quick Run Card
3. Open Session Notes
4. Ask the user the main problem


OPENING QUESTIONS
- What feels slow?
- What do you use this PC for most?
- Is anything not working properly?
- Are there files you are worried about losing?


FAST DEFAULT PATH
1. Run PowerShell Disk Space Audit
2. Run PowerShell Downloads Folder Audit
3. Clean obvious junk from Downloads
4. Tidy obvious Desktop clutter
5. Empty Recycle Bin
6. Recheck free space
7. Write Session Notes


ONLY GO FURTHER IF NEEDED
- If startup is slow → run Startup Items Review
- If the PC feels cluttered/confusing → run Installed Programs Export
- If space is still tight → run Large Files Finder
- If the complaint is browsing/pop-ups → use Browser Triage Guide


SAFE WIN ORDER
1. obvious junk in Downloads
2. obvious Desktop clutter
3. Recycle Bin
4. careful app review
5. careful startup review
6. careful browser review


STOP RULES
Stop if:
- you are unsure what it is
- it looks important
- it looks system-related
- it looks security-related
- it looks printer-related and she still uses the printer
- the risk is higher than the likely gain


DO NOT TOUCH
- system folders
- unknown files
- security software
- printer software unless clearly unnecessary
- saved passwords or browser accounts without asking


WIN CONDITION
- more free space if storage was the issue
- less clutter
- clearer understanding of the machine
- no accidental damage
- good notes for next time


END OF SESSION
- check whether the main complaint improved
- recheck free space if relevant
- save Session Notes
- write the next best step


RULE
Take the clearest safe win first.
If unsure, leave it and write it down.
