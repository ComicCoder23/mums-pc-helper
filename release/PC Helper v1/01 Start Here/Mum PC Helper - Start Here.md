MUM PC HELPER — START HERE


Purpose
Mum PC Helper is a safe first-pass Windows support kit for home and family PCs.


What it helps with
- slow PCs
- cluttered desktops and downloads
- low disk space
- startup clutter
- confusing home PC setups
- light browser clutter checks


What it is not
- not an aggressive cleanup tool
- not a deep repair suite
- not a promise to fix every PC problem in one visit


Best way to use it
1. Read the Quick Run Card
2. Read the Visit Workflow One Pager
3. Use the Action Decision Tree if the problem is unclear
4. Read the User Guide for fuller instructions
5. Use the PowerShell scripts carefully


Core rule
Take the clearest safe win first.
If unsure, leave it alone.


What comes with Mum PC Helper
- Quick Run Card
- Visit Workflow One Pager
- Action Decision Tree
- User Guide
- Safety Notes
- Script bundle
