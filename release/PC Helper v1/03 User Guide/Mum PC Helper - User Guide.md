MUM PC HELPER — USER GUIDE


Purpose
This guide explains how to use Mum PC Helper safely as a simple first-pass Windows PC support product.


What this product is
Mum PC Helper is a guided support kit for checking a slow or cluttered Windows PC and taking the clearest safe actions first.


What this product is not
- not a deep repair suite
- not an aggressive cleanup tool
- not a malware removal toolkit
- not a promise to fix every PC problem in one visit


Main principle
Take the clearest safe win first.
If unsure, leave it alone.


Before you start
Ask:
- What feels slow?
- What do you use this PC for most?
- Is anything not working properly?
- Are there files you are worried about losing?
- Are there things you do not want touched?


Best opening order
1. Start Here
2. Quick Run Card
3. Visit Workflow One Pager
4. Action Decision Tree
5. Safety Notes


Default safe path
1. Run Disk Space Audit
2. Run Downloads Audit
3. Clean obvious junk from Downloads
4. Tidy obvious Desktop clutter
5. Empty Recycle Bin
6. Recheck free space


Only go further if needed
- If startup is slow, review startup items
- If there are many unknown apps, export installed programs
- If space is still tight, run the large files finder


Safe cleanup order
1. Downloads
2. Desktop
3. Recycle Bin
4. Documents only if needed
5. Pictures and Videos only if needed
6. Installed apps only if needed
7. Startup items only if needed


What not to touch casually
- system folders
- unknown files
- printer software
- security software
- Windows-related components
- browser passwords or account data without asking


End of session
- check whether the main complaint improved
- recheck free space if storage was the issue
- note what was changed
- leave uncertain items alone


Final rule
Audit first.
Delete obvious junk only.
If unsure, leave it and write it down.
