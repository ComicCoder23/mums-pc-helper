MUM PC HELPER — RESULTS INTERPRETATION GUIDE


Purpose
This guide helps you read the script reports quickly and decide what to do next without overthinking or making risky changes.


MAIN DECISION RULE
Read the report.
Identify the biggest likely cause.
Take the safest obvious action first.
Stop when the next step becomes uncertain.


1. DISK SPACE AUDIT — HOW TO READ IT


If free space is low:
- likely cause = storage clutter
- first action = check Downloads and Desktop
- second action = remove obvious junk only
- do not jump into system cleanup


If one user folder is much larger than the others:
- likely cause = clutter concentrated in one location
- first action = inspect that folder carefully
- if it is Downloads, use the Downloads Audit report
- if it is Pictures or Videos, ask before touching anything


If space is not low:
- storage may not be the main problem
- shift attention to startup clutter, app clutter, browser issues, or general slowness


2. DOWNLOADS AUDIT — HOW TO READ IT


If there are many old installers and archive files:
- likely cause = easy cleanup opportunity
- first action = review old installers with Mum
- safe wins = outdated installers, duplicate downloads, extracted zip leftovers


If there are many large unknown files:
- likely cause = clutter, but risk is higher
- first action = ask before deleting
- if uncertain, leave them and note them down


If Downloads is small and clean:
- Downloads is not the main issue
- move on to Desktop, installed apps, or startup items


3. INSTALLED PROGRAMS EXPORT — HOW TO READ IT


If there are lots of programs Mum does not recognize:
- likely cause = app clutter or years of buildup
- first action = mark unknown apps for discussion
- do not uninstall on the spot unless the app is clearly unnecessary


If there are duplicate-looking apps, old toolbars, or obvious junk:
- likely cause = app clutter
- first action = review with Mum and remove only what is clearly safe


If most apps look normal and familiar:
- installed programs are probably not the main issue
- move on to startup or browser checks


4. STARTUP ITEMS REVIEW — HOW TO READ IT


If there are many startup items:
- likely cause = startup slowness
- first action = review in Task Manager
- disable only obvious non-essential items


If startup items include security, printer, sound, sync, or system items:
- do not disable casually
- leave them unless you are sure


If startup list is short:
- startup clutter probably is not the main problem
- the issue may be storage, browser, or general PC age


QUICK DECISION MAP


Case A — Low space + messy Downloads
Action:
- clean Downloads first
- tidy Desktop second
- empty Recycle Bin
- recheck space


Case B — Space fine but startup slow
Action:
- review startup items
- use Task Manager carefully
- do not disable unknown system-related items


Case C — Space fine but PC feels cluttered and confusing
Action:
- export installed programs
- discuss what is actually used
- remove only clearly unnecessary apps


Case D — Reports show no obvious problem
Action:
- do not force cleanup
- note what was checked
- ask more about browser, internet, or specific app slowness


WHEN TO STOP
Stop if:
- the next step is uncertain
- the item looks important
- the app looks system-related
- the file is personal and not clearly junk
- the gain from continuing looks small compared with the risk


SAFE PRIORITY ORDER
1. Obvious junk in Downloads
2. Obvious Desktop clutter
3. Recycle Bin
4. Review apps carefully
5. Review startup carefully
6. Leave uncertain items alone


WHAT SUCCESS LOOKS LIKE
- obvious junk removed
- more free space if storage was the issue
- clearer understanding of the machine
- no accidental damage
- good notes for next time


RULE
Take the clearest safe win first.
Do not keep digging once the easy evidence runs out.
