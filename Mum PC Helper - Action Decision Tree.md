PC HELPER — ACTION DECISION TREE


Purpose
This is the fastest decision guide in the pack.
Use it when you want to move quickly without overthinking.


START


1. Ask the user what the main problem is.
Then choose the closest path below.


PATH A — "The PC is full / no space left"
Do this:
1. Run PowerShell Disk Space Audit
2. Run PowerShell Downloads Folder Audit
3. Clean obvious junk from Downloads
4. Tidy obvious Desktop clutter
5. Empty Recycle Bin
6. Recheck free space


If still bad:
- review Pictures/Videos carefully
- only if the user agrees


Do not:
- touch system folders
- mass delete unknown files


PATH B — "The PC takes ages to start"
Do this:
1. Run PowerShell Startup Items Review
2. Open Task Manager Startup tab
3. Review obvious non-essential startup items
4. Leave unknown, printer, sound, security, and system items alone


If still bad:
- check installed programs next


Do not:
- disable things you do not understand


PATH C — "The PC feels cluttered and confusing"
Do this:
1. Open First Visit Checklist
2. Check Desktop
3. Check Downloads
4. Run PowerShell Installed Programs Export if needed
5. Record unknown apps in Session Notes


If there is obvious junk:
- clean the easy wins first


Do not:
- uninstall apps just because the name looks unfamiliar


PATH D — "I keep getting pop-ups / something seems wrong"
Do this:
1. Stop normal cleanup
2. Record the exact behavior in Session Notes
3. Check browser extensions and obvious browser clutter
4. Use Malwarebytes only if needed and available


Do not:
- start deleting random files
- assume the issue is just low space


PATH E — "I can never find anything"
Do this:
1. Check Desktop clutter
2. Check Downloads clutter
3. Ask what folders the user actually uses
4. Move only obvious misplaced files
5. Write down what was moved


Do not:
- reorganize the whole PC in one visit


PATH F — "I do not know, it is just slow"
Do this in order:
1. Ask what feels slow exactly
2. Run Disk Space Audit
3. Run Downloads Folder Audit
4. If startup is mentioned, run Startup Items Review
5. If the PC feels cluttered, run Installed Programs Export


Then take the clearest safe win first.


FAST STOP RULES
Stop and leave it alone if:
- you are unsure what it is
- it looks important
- it looks system-related
- it looks printer-related and she still prints
- it looks security-related
- the gain is small but the risk is high


SAFE WIN ORDER
Always prefer this order:
1. obvious junk in Downloads
2. obvious Desktop clutter
3. Recycle Bin
4. careful app review
5. careful startup review


SESSION END
Before finishing:
- check whether the main complaint improved
- recheck free space if space was the issue
- save Session Notes
- write the next best step


RULE
Take the clearest safe win.
Leave uncertain things for later.
